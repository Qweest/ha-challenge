# Project overview

Expo SDK 57 loyalty challenge app using Expo Router, TypeScript, NativeWind, TanStack Query, Axios, and Jotai. Source code lives in `src/`.

- Use Bun commands from `package.json`; run `bun lint` and `bunx tsc --noEmit` before finishing.
- Keep implementations simple, filenames in kebab-case, and use `cn` from `src/helpers.ts` when merging classes.
- Consult the exact [Expo SDK 57 documentation](https://docs.expo.dev/versions/v57.0.0/) before changing Expo behavior.
